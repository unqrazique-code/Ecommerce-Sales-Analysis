# E-Commerce Sales Analysis

## Overview

This project analyzes e-commerce sales data to uncover trends, evaluate business performance, and generate actionable insights. The analysis is supported by an interactive Power BI dashboard for clear and effective visualization.

---

## Objectives

* Analyze sales trends over time  
* Identify top-performing products and categories  
* Evaluate region-wise sales performance  
* Understand profit distribution and key metrics  

---

## Tools Used

* Python (pandas, matplotlib)  
* Power BI  

---

## Dashboard Features

* Monthly Sales Trend (Line Chart)  
* Top-Selling Products (Bar Chart)  
* Region-wise Sales Analysis (Map/Bar Chart)  
* Profit Distribution (Chart)  
* KPI Cards (Total Sales, Total Profit)  

---

## Key Insights

* Sales show a steady trend over time  
* Technology products generate the highest revenue  
* The West region contributes the maximum sales  
* Profit varies across different product categories
