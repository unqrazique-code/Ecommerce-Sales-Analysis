# E-commerce Sales Data Analysis

##  Objective
Analyze sales data to identify trends, top products, and regional performance.

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI

## Dashboard Features
- Monthly sales trend
- Top-selling products
- Region-wise sales analysis
- Profit distribution
- KPI metrics (Total Sales, Total Profit)

##  Key Insights
- Sales show a steady trend over time
- Technology products generate highest revenue
- West region contributes maximum sales
- Profit varies across categories

## Project Structure
- data/ → dataset
- notebooks/ → analysis notebook
- dashboard/ → Power BI dashboard
